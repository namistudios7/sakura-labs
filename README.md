## How to Deploy

### Option 1: Vercel (Easiest)
1. Go to vercel.com
2. Click "New Project"
3. Import this repository (or drag-and-drop the folder)
4. Deploy (takes 30 seconds)

### Option 2: Netlify
1. Go to netlify.com
2. Click "New site from Git" or drag-and-drop the folder
3. Deploy (automatic)

### Option 3: Any Static Host
- Upload `index.html` to any web hosting
- Works with GitHub Pages, AWS S3, etc.
